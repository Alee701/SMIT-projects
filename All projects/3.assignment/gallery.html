MangoDB

MongoDB is a popular open-source NoSQL database management system that falls under the category of document-oriented databases. It was developed by MongoDB Inc. and is designed to handle large volumes of data, particularly unstructured or semi-structured data.

Express JS

Express.js, commonly referred to as Express, is a minimal and flexible web application framework for Node.js. It provides a robust set of features to develop web and mobile applications. Express is widely used for building server-side applications and APIs (Application Programming Interfaces) using JavaScript.

React

React, developed and maintained by Facebook, is a popular open-source JavaScript library for building user interfaces or UI components. It is often used for building single-page applications (SPAs) where user interactions are dynamic and the UI needs to be updated in real-time.
